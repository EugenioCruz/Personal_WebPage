# Photography Portfolio

Add your photos here and update `photography.html` to display them.

## Structure

Place your images in this folder and reference them in the gallery section of `photography.html`.

## Example usage in photography.html:

```html
<div class="photo-card" data-category="mountains">
    <img src="photos/your-photo.jpg" alt="Description">
    <div class="photo-overlay">
        <div class="photo-title">Photo Title</div>
        <div class="photo-location">Location</div>
    </div>
</div>
```

## Categories available:
- mountains
- climbing
- travel
- nature

You can add more categories by updating the filter buttons in photography.html
